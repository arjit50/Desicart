# ⚡ Page Transitions - Quick Start

## 🚀 Test It Right Now

### Option 1: Demo Mode (Best for testing all 5)
1. Go to: `http://localhost:8000/?test_transitions=true`
2. Click the buttons in the top-right panel to switch transitions
3. Click any link to see the active transition

### Option 2: Just Start Navigating
Click around your site - the transition plays on every page change!

---

## 5 Animations Available

```
🔲 FADE              → Simple fade out/in
⬆️  SLIDE UP         → Modern upward slide
✨ BLUR & FADE      → Premium (CURRENTLY ACTIVE) ⭐
📦 SCALE & FADE     → Content shrinks
➡️  SLIDE RIGHT     → Directional slide
```

---

## Change Active Transition (30 seconds)

1. Open: `resources/js/page-transitions.js`
2. Find: `const ACTIVE_TRANSITION = 'blur-fade';` (line 7)
3. Replace with one of:
   - `'fade'`
   - `'slide-up'`
   - `'blur-fade'` ← Recommended
   - `'scale-fade'`
   - `'slide-right'`
4. Refresh browser

That's it! ✓
