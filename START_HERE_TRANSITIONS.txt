## 🔄 How to Change Active Transition

It's only **one line to change**!

1. Open file:
   ```
   resources/js/page-transitions.js
   ```

2. Find line 7:
   ```javascript
   const ACTIVE_TRANSITION = 'blur-fade';
   ```

3. Replace with any of these:
   ```javascript
   const ACTIVE_TRANSITION = 'fade';           // Try this for minimal feel
   const ACTIVE_TRANSITION = 'slide-up';       // Try this for modern feel
   const ACTIVE_TRANSITION = 'blur-fade';      // Current (recommended)
   const ACTIVE_TRANSITION = 'scale-fade';     // Try this for focus
   const ACTIVE_TRANSITION = 'slide-right';    // Try this for dynamic feel
   ```

4. Save and refresh browser

Done! ✓

## 🎯 Excluded from Transitions

The system is smart and does NOT transition for:
- External website links
- Links opening in new tabs/windows
- Logout button
- Form submissions
- Anchor links (#)
- Any link with `data-no-transition` attribute

This ensures smooth UX and no unwanted interruptions.
