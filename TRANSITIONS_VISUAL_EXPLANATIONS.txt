# 🎬 Page Transitions - Visual Descriptions

## How They Look (ASCII Art + Description)

---

### 1️⃣ FADE Transition
```
 BEFORE          DURING              AFTER
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   Page A    │  │   FADING    │  │   Page B    │
│ All content │→ │ (White in)  │→ │ All content │
│   visible   │  │ (fades out) │  │   visible   │
└─────────────┘  └─────────────┘  └─────────────┘

Timeline: 0% ────→ 50% ────→ 100%
Opacity:  100% → 0% (white) → 100%

Effect: Page disappears, white flash, new page appears
Feel: Clean and minimal
```

---

### 2️⃣ SLIDE UP Transition
```
         SLIDE UP EFFECT
         ↑ ↑ ↑ ↑ ↑ ↑
┌─────────────────────┐
│    NEW PAGE B       │ ← Slides up from bottom
│   Coming up...      │
├─────────────────────┤
│    PAGE A           │ ← Still visible, gets replaced
│   Stays visible     │
└─────────────────────┘

Timeline: 0% ────→ 50% ────→ 100%
Position: Bottom → Middle → Top

Effect: Content slides upward smoothly
Like turning a page in a book
Feel: Modern and smooth
```

---

### 3️⃣ BLUR & FADE Transition ⭐ RECOMMENDED
```
 CLEAR          BLURRING            CLEAR AGAIN
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   Page A    │  │   Page A    │  │   Page B    │
│ All content │→ │ (BLURRED)   │→ │ All content │
│   visible   │  │ (fading)    │  │   visible   │
│             │  │ White over  │  │             │
└─────────────┘  └─────────────┘  └─────────────┘

Timeline: 0% ────→ 50% ────→ 100%
Blur:     0px → 12px → 0px
Opacity:  100% → full white → 100%
White:    transparent → opaque → transparent

Effect: Page blurs and fades with white overlay
Then clears to show new page
Feel: Premium, elegant, glassmorphic
✨ BEST FOR YOUR SITE ✨
```

---

### 4️⃣ SCALE & FADE Transition
```
 NORMAL SIZE      SHRINKING         NORMAL SIZE
┌─────────────┐   ┌───────────┐   ┌─────────────┐
│             │   │           │   │             │
│  Page A     │   │  Page A   │   │  Page B     │
│ 100% size   │→  │ 95% size  │→  │ 100% size   │
│             │   │ (fading)  │   │             │
│             │   │ GREEN     │   │             │
└─────────────┘   └───────────┘   └─────────────┘

Timeline: 0% ────→ 50% ────→ 100%
Scale:    100% → 95% → 100%
Opacity:  100% → 0% (white) → 100%
Tint:     none → green → none

Effect: Page shrinks slightly while fading
Like zooming out then back in
Feel: Focused and centered
```

---

### 5️⃣ SLIDE RIGHT Transition
```
STARTING       SLIDING RIGHT       FINISHED
Page A         ─────────────→      Page B
┌──────┐       ┌──────┐           ┌──────┐
│ Page │───→   │ Page │───→       │ Page │
│  A   │  \    │  A   │  \        │  B   │
└──────┘   \   └──────┘   \       └──────┘
           │White Overlay │
           │with gradient │

Timeline: 0% ────→ 50% ────→ 100%
Position: Left → Middle → Right
Gradient: slides across screen

Effect: Page slides right with white gradient overlay
Like swiping to the next page
Feel: Dynamic and directional
```

## How to Switch

Edit this ONE line in `resources/js/page-transitions.js`:
```javascript
const ACTIVE_TRANSITION = 'blur-fade';
```

Change to any of:
```javascript
const ACTIVE_TRANSITION = 'fade';
const ACTIVE_TRANSITION = 'slide-up';
const ACTIVE_TRANSITION = 'blur-fade';      ← Current
const ACTIVE_TRANSITION = 'scale-fade';
const ACTIVE_TRANSITION = 'slide-right';
```

Refresh your browser. That's it!
