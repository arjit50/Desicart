## 🎬 HOW IT WORKS

```
User navigates the site
        ↓
Script detects link click
        ↓
Check if it's an internal link (not external, not new tab)
        ↓
Show page-transition-overlay
        ↓
Add 'page-transition-active' class
        ↓
CSS animation plays (500ms)
    - BLUR & FADE: blurs and fades to white
    - Other options: other effects
        ↓
Navigate to new page
        ↓
New page loads
        ↓
Animation is automatically reset
        ↓
Smooth experience! ✨
```

## ❓ FAQ

**Q: Will this slow down my site?**
A: No! GPU-accelerated animations are smooth 60fps.

**Q: Works on all devices?**
A: Yes! Desktop, tablet, mobile - all perfect.

**Q: Can I change the timing?**
A: Yes! Edit duration in `resources/css/theme.css`.

**Q: Can I disable on certain links?**
A: Yes! Add `data-no-transition` to any link.

**Q: Which one should I use?**
A: BLUR & FADE (currently active, already perfect!)
